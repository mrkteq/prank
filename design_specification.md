# Project Design Specification: Intentionally Terrible Portfolio Website

## Project Overview
A single-page website that demonstrates AI prompting and direction skills through deliberate anti-design. The site showcases the ability to guide an LLM to create purposefully bad UX/UI while maintaining a coherent creative vision.

**Target Audience:** Potential employers, collaborators, or anyone interested in AI collaboration and creative prompting skills.

**Core Concept:** A hilariously terrible website that transforms into a professional portfolio, proving the chaos was intentional and strategic.

---

## Typography Specifications

### Primary Font Pairing (Intentionally Horrible)
- **Headings:** Comic Sans MS
  - All major section titles
  - Hero name/title
  - Navigation elements (if any)
  
- **Body Text:** Papyrus
  - All paragraph content
  - Descriptions
  - Form labels

- **Accent/Special Text:** Mix randomly between:
  - Impact (for "important" callouts)
  - Brush Script MT (for testimonials)
  - Courier New (for "code" references)
  - Times New Roman (sprinkled randomly)

### Typography Rules
- Font sizes should vary wildly (12px to 72px with no logical hierarchy)
- Random use of ALL CAPS, Title Case, and lowercase
- Mix bold, italic, and underline unpredictably
- Letter spacing should be uncomfortable (too tight or too loose)
- Line height should create awkward text blocks

## Page Structure & Sections

### 1. Hero Section
**Content Requirements:**
- Name in 3+ different fonts simultaneously
- Job title: "AI Whisperer / Web Developer / Prompt Engineer Extraordinaire"
- Animated elements: spinning text, bouncing elements
- Under construction GIF (classic yellow/black)
- Visitor counter (broken, stuck at 000042)
- Marquee text: "WELCOME TO MY WEBSIT - UNDER CONSTRUCTION SINCE 2024"
- Auto-play background MIDI music (if technically feasible)

**Visual Elements:**
- Clashing neon background (hot pink with lime green gradients)
- Tiled background pattern (starfield or similar)
- Lens flare effects
- Multiple animated GIFs (dancing baby, flame borders, spinning @ symbol)

---

### 2. About Me Section
**Content Requirements:**
- Wall of text with minimal paragraph breaks
- Intentional typos and grammar mistakes scattered throughout
- TMI personal details ("I have 3 cats and eat cereal for dinner sometimes")
- Vague professional claims mixed with absurd qualifications
- Include phrases like "synergy," "paradigm shift," "thinking outside the box"
- Random capitalization of ordinary words

**Visual Elements:**
- Low-resolution pixelated profile photo with rainbow border
- "Currently listening to:" widget (broken)
- Mood ring indicator
- Blinking text effects
- Starburst "NEW!" badges scattered randomly

### 3. My Skills Section
**Content Requirements:**
- Skill list including:
  - AI Prompting (150%)
  - Microsoft Word (200%)
  - Typing (87%)
  - HTML (maybe)
  - Synergy Generation (999%)

**Visual Elements:**
- Animated progress bars that exceed 100%
- Bars moving at different speeds
- Conflicting colors for each skill bar

### 4. Portfolio/Projects Section

- 6-8 project cards

**Visual Elements:**
- Image gallery with broken images
- Inconsistent card sizes
- Random rotation angles on cards
- Drop shadows going different directions
- "Featured" ribbons on every single project
- Sparkle cursor trail effects

### 5. Testimonials Section
**Content Requirements:**

- Names like:
  - "John Business, CEO of Business Corp"

  - "Jane Professional, Industry Expert"

**Visual Elements:**
- Stock photo faces (obviously watermarked or mismatched)
- Star ratings that are different sizes
- Quote marks that don't match (opening quote used twice)
- Comic speech bubbles pointing wrong directions
- Animated testimonial carousel that moves too fast

### 6. Contact Section
**Content Requirements:**
- Form with absurd required fields:
  - Name
  - Email
  - Mother's Maiden Name's Favorite Color
  - Social Security Number (for verification purposes)
  - Quest to Find the Holy Grail
  - Favorite Pokemon (Gen 1 only)
  - Captcha: "What is 2+2?" with dropdown showing 1, 3, 5, 22, Purple

- Spinning envelope GIF

---

### 7. The Secret Reveal
**Trigger Mechanism:**
- Tiny button hidden in footer: "Make it Stop" or "Fix This"
- Alternative: Konami code implementation
- Alternative: Click the visitor counter 3 times

**Transformation:**
- Entire page smoothly transitions to clean, minimal, professional design
- Reveals actual portfolio content
- Shows documentation of the prompting process
- Explains the project concept
- Links to real work and contact information

**Clean Version Should Include:**
- Professional bio
- Real project showcase
- Actual contact form
- Documentation: "How I Prompted AI to Create This Monstrosity"
- Screenshots of prompts and AI responses
- Reflection on AI collaboration

## Color Palette (Intentionally Clashing)

**Primary Colors:**
- Hot Pink (#FF69B4)
- Lime Green (#00FF00)
- Electric Blue (#0000FF)
- Safety Orange (#FF8C00)
- Cyan (#00FFFF)

**Background Colors:**
- Multiple tiled background images
- Gradient overlays that clash
- Random section background colors with no consistency

**Text Colors:**
- Should be difficult to read against backgrounds
- Random color changes mid-sentence
- Links in red instead of blue
- Visited links in same color as unvisited

## Interactive Elements & Animations

### Required Animations:
- Marquee text (scrolling in multiple directions)
- Blinking text (via CSS animation)
- Rotating elements (skill icons, profile photo)
- Bouncing buttons
- Particle effects (sparkles, stars)
- Cursor trails
- Pop-ups that spawn more pop-ups

### User Interaction Patterns (Anti-UX):
- Buttons that move on hover
- Forms that reset when you mistype
- Links with misleading labels
- Scroll hijacking (if subtle enough to be funny, not rage-inducing)
- Right-click disabled with "Nice try!" alert
- Auto-refreshing sections
- Progress bars that go backwards

---

## Technical Requirements

### Must Have:
- Single HTML page (can have CSS and JS files)
- Responsive (but breaks hilariously on mobile)
- Fast load time despite appearing broken
- Accessible transformation button (for the reveal)
- Works in modern browsers
- No actual data collection (fake form)

### Nice to Have:
- Local storage to remember if user found the transformation
- Konami code Easter egg
- Sound effects on clicks (annoying)
- Print stylesheet that makes it worse
- Meta tags that are joke descriptions

## Content Tone & Voice

**Terrible Version:**
- Overly enthusiastic with excessive punctuation!!!
- Corporate buzzword salad
- Inconsistent perspective (switching between first and third person)
- Random tangents
- Obvious typos left in intentionally
- Self-aggrandizing but vague

**Professional Version (After Transformation):**
- Clear, confident, competent
- Explains the artistic/strategic vision
- Documents the AI collaboration process
- Demonstrates actual technical and creative skills

## Success Metrics

This project successfully demonstrates:
1. **AI Direction Skills:** Every terrible choice is documented as intentional
2. **Technical Competence:** The transformation proves you can build proper websites
3. **Creative Vision:** The cohesive awfulness shows strategic thinking
4. **Humor & Self-Awareness:** The concept itself is the portfolio piece

---

## Implementation Notes for Developer

When prompting the LLM to build this:
- Be specific about which elements should be terrible and how
- Request inline CSS for maximum chaos (though external sheet is fine)
- Ask for commented code explaining the intentional anti-patterns
- Ensure the transformation mechanism is smooth and impressive
- Keep the reveal content professional and genuinely portfolio-worthy
- Test that the terrible version is funny, not actually frustrating
